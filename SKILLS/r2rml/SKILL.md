# R2RML 匹配技能

## 使用方式
- 以“本体数据属性”为锚点，为每个属性选择一个最合适的表字段。
- 综合考虑：属性名语义、字段名语义、样例数据类型、属性的 domain/range 关系。
- 输出必须引用已提供的表名与字段名；如果没有合适匹配，返回空值。

## 匹配策略
- 优先比较属性名与字段名的相似度。
- 使用样例数据判断字段类型（数字、日期、邮箱、URL、电话等），与属性 range 或语义关键词比对。
- 若属性有 domain 信息，将表名与 domain 名称做相似度补充。

## 输出格式
- 输出 JSON：
  - `matches`: 每个属性对应一个匹配项
  - 每个匹配项包含：`property_iri`、`table_name`、`field`
- 无匹配则将 `table_name` 与 `field` 设为 `null`。
